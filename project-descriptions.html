Unity 2D Story Platformer Game

A charming 2D story-driven platformer developed with Unity. This game combines puzzle-solving mechanics with a heartfelt narrative and hand-drawn visuals.

Key Features:

  • Side-scrolling platforming gameplay
  • Story-rich progression with dialogue scenes
  • Physics-based puzzles
  • Unlockable abilities and collectibles
  • Original soundtrack and sound effects

Technologies:

  • Unity Engine, C#
  • Adobe Photoshop and Illustrator for 2D art
  • Cinemachine for camera control
  • FMOD for audio design
  • Git for version control
